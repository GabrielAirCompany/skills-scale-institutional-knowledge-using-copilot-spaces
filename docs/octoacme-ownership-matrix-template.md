# OctoAcme — Project Ownership Matrix Template

## Purpose
Provide a lightweight RACI-style reference so every team member knows who is Responsible, Accountable, Consulted, and Informed for each key activity in a project.

## How to use
1. Copy this template into your project's `docs/` folder (or link it from the Project One-pager).
2. Fill in the **Activity** column with the major work streams or decisions for your project.
3. Assign a RACI value for each role column. Use only one **A** (Accountable) per row.
4. Review and agree on the matrix at project kickoff. Update as the project evolves.

## RACI Key
| Code | Meaning |
|------|---------|
| **R** | **Responsible** — does the work |
| **A** | **Accountable** — owns the outcome; single decision-maker per activity |
| **C** | **Consulted** — provides input before a decision or action |
| **I** | **Informed** — kept up to date on progress or outcome |
